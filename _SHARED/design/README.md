# Creative Design + Ergonomics

> **Department type:** Active
> **Owner:** Hazza (Design Lead)
> **Division:** Design & Frontend

---

## Purpose

This department owns the "why" behind every design decision. It is responsible for UX philosophy, age-appropriate design principles, content systems (including the Nendō), and the creation of new skills and workflows.

Where UI & Prototypes builds what users see, this department decides what they *should* see — and why.

## Scope

**In scope:**
- UX philosophy and design rationale
- Age-appropriate design standards (Ninja Cubs 3–5, Ninja School 5–10, 11+)
- Content system design for the Nendō (challenges, progression, Aura system)
- Ergonomics: layout, readability, cognitive load, accessibility for young users
- Creating and maintaining skills for all departments via `skill-creator`
- Working Doc authorship and updates

**Out of scope:**
- Building HTML prototypes (→ UI & Prototypes)
- Schema or API implementation (→ Backend & Deployment)
- Server configuration (→ Infrastructure & Security)

## Key Domains

### Age Design
Ninja School serves three distinct age groups. Every screen, interaction, and piece of content must be appropriate for its audience. This department sets the standards:

- **Ninja Cubs (3–5):** Large touch targets, minimal text, visual/audio feedback, narrative-led ("Ninja Narrative")
- **Ninja School (5–10):** Core audience. Balanced literacy, gamified progression, challenge-driven engagement
- **11+ (Secondary):** More structured information, reduced gamification, increased autonomy

### Nendō Content
The Nendō (Stage 2) is a digital extension of Ninja School's mindfulness philosophy. This department owns the content design:

- Challenge structure and progression
- Aura system (replaces belt language — belts belong to the physical dojo)
- Mushin practice design (active stillness, never passive or boring)
- All content must be appropriate for 7–9 year olds

### Skill Creation
This department is the factory for new skills. When any department needs a new specialist capability, Creative Design + Ergonomics designs, tests, and deploys it using the `skill-creator` skill.
